# Portfolio Template

Copy `page.tsx` into a route or screen component in a project that already:

- installs `@weiyinblue/ui`
- imports `@weiyinblue/ui/styles.css`
- mounts `ThemeProvider`
- has `lucide-react` available

This template is intended for:

- portfolio landing pages
- product overview intros
- personal site hero-to-case-study flows
